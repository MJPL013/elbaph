# Elbaph Texture Prompt Library

No images are generated in this phase. Each numbered prompt is the **Base Prompt**
followed by its category-specific instruction.

## Base Prompt

> Use case: stylized-concept. Asset type: reusable WebGL game albedo texture for
> Elbaph, a cozy handcrafted low-poly spherical world. Style: hand-painted indie
> game texture, soft pastel palette, clean readable shapes, restrained brush
> variation, slightly imperfect painted edges. Camera: flat orthographic surface,
> no perspective. Lighting: neutral albedo only; no baked lighting, highlights,
> cast shadows, ambient occlusion, reflections, normals, depth, or metallic shine.
> Constraints: not photorealistic, not busy, no watermark, no text unless the
> category explicitly requests a reserved sign area. Keep edge contrast suitable
> for three-band cel shading and thick ink outlines. For tileable surfaces, make
> every edge perfectly seamless.

## Authoring and Runtime Rules

- Generate source masters as PNG, then review seams and remove any accidental lighting.
- Pack approved sources into category atlases with 16-pixel gutters.
- Use KTX2 ETC1S for opaque atlases and KTX2 UASTC for alpha-sensitive decals.
- Provide WebP fallbacks and 512-pixel low-quality variants.
- Add exact company names and technical labels after generation using deterministic SVG/canvas composition.
